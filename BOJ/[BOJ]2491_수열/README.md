## algorithm 💻
1. 현재 값이 직전 값보다 크면 커지는 수열의 크기 증가, 작아지는 수열 1로 초기화
    ```java
    if (arr[i - 1] < arr[i]) {
        big++;
        small = 1;
    }
    ```

2. 현재 값과 직전 값이 일치하면 커지는 수열, 작아지는 수열의 값을 증가
    ```java
    else if (arr[i - 1] == arr[i]) { 
        big++;
        small++;
    }
    ```

3. 현재 값이 직전 값보다 작으면 작아지는 수열의 크기 증가, 커지는 수열 1로 초기화
    ```java
    else {
        small++;
        big = 1;
    }
    ```

4. 중간 중간에 수열의 최대값을 체크해 갱신해주면 끝!
