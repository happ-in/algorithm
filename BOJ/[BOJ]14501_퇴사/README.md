#### 2. 벌 수 있는 최대 금액을 저장, 비교 😍
```{.java}
dp[i] = Math.max(max, dp[i]);
dp[i + time[i]] = Math.max(dp[i + time[i]], dp[i] + cost[i]);
max = Math.max(max, dp[i]);
```

* 오늘 최대 금액 = 최대값(전 날까지의 max 금액, 오늘 금액)
* 오늘 상담해서 끝나는 + T[i] 일의 최대 금액 = 최대값(i + T[i] 날의 최대 금액, 오늘 날짜 + P[i])
* max 값 갱신
